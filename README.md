# Discipline Tracker (Python)

A small CLI tool to track your daily study discipline (hours per day) with safe input handling.

## Features
- Asks how many hours you study per day
- Returns a level:
  - Beginner (progress)
  - Good level
  - Elite level
- Safe loop:
  - `q` to quit
  - prevents invalid inputs
  - safety guard against uncontrolled infinite loops

## Requirements
- Python 3.x

## Run
```bash
python discipline_tracker.py
